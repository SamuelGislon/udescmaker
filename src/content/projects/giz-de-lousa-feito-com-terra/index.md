Este projeto transforma materiais simples em uma atividade maker acessível para oficinas, escolas e espaços comunitários. A proposta aproxima crianças e jovens de processos de experimentação, mostrando que é possível criar ferramentas úteis a partir de recursos cotidianos.

O passo a passo foi pensado para ser reproduzido com facilidade, em pequenos grupos, incentivando conversa sobre textura, secagem, reaproveitamento e observação de resultados. O resultado final rende um giz com aparência artesanal, pronto para mural, calçada ou lousa improvisada.
